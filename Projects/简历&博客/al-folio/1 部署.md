环境：wsl ubuntu 24
### 安装依赖
```
# 更新包信息
sudo apt update

# 安装 curl（如果没有）
sudo apt install curl -y

# 添加 Node.js 18.x 官方源（可根据需要替换版本）
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -

# 安装 Node.js（包含 npm）
sudo apt install -y nodejs

# 检查版本
node -v
npm -v

```
