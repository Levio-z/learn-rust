### IRQ 对应设备功能解析

Intel 8259 的每条 IRQ 对应一个外设或功能线，下面详细解释每条中断的用途：

| IRQ | 设备（示例）                    | 功能说明                                                                   |
| --- | ------------------------- | ---------------------------------------------------------------------- |
| 0   | Timer                     | 系统定时器（PIT, Programmable Interval Timer），定时中断，用于时间片调度、系统时钟更新。           |
| 1   | Keyboard                  | 键盘控制器中断，当按键按下或释放时触发，用于读取按键扫描码。                                         |
| 2   | Cascade（连接 Secondary PIC） | 用于连接辅助 PIC 的中断线，实现主从级联扩展 IRQ。主 PIC 的 IRQ2 是“门”，从 PIC 的 IRQ 通过它传递给 CPU。 |
| 3   | Serial Port 2             | 第二串口（COM2）数据传输中断，用于串口通信。                                               |
| 4   | Serial Port 1             | 第一串口（COM1）数据传输中断。                                                      |
| 5   | Parallel Port 2/3         | 并口（LPT2 或 LPT3）通信中断，例如打印机。                                             |
| 6   | Floppy Disk               | 软盘控制器中断，用于软盘读写完成通知。                                                    |
| 7   | Parallel Port 1           | 主并口（LPT1）通信中断。                                                         |
| 8   | RTC (Real Time Clock)     | 实时时钟中断，用于周期性唤醒系统、唤醒事件或闹钟功能。                                            |
| 9   | ACPI / 可用                 | 高级配置与电源接口（ACPI）或其他可编程用途。                                               |
| 10  | 可用                        | 用户或扩展设备可编程中断。                                                          |
| 11  | 可用                        | 同上，可供外设自定义中断使用。                                                        |
| 12  | Mouse                     | PS/2 鼠标中断，用于鼠标移动和点击事件。                                                 |
| 13  | Co-Processor              | 数学协处理器（FPU）异常中断，用于浮点运算错误或状态通知。                                         |
| 14  | Primary ATA               | 主 ATA 硬盘控制器中断，读写完成或错误通知。                                               |
| 15  | Secondary ATA             | 辅助 ATA 硬盘控制器中断，同上                                                      |
