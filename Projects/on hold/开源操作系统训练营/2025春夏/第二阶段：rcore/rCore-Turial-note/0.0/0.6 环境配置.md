### Docker

```rust
git clone https://github.com/LearningOS/rCore-Tutorial-Test-2025S.git user
```


```rust
# 1. 切换到 root 用户（如果未设置 root 密码，先设置）
sudo passwd root
su root

# 2. 进入 rCore-Tutorial-v3 根目录
cd rCore-Tutorial-v3

# 3. 构建基于 Docker 的开发环境
make build_docker

# 4. 启动并进入 Docker 环境
make docker

# 5. 在 Docker 环境中：切换到 /mnt
cd /mnt

# 6. 查看 /mnt 下的内容（与 rCore-Tutorial-v3 相同）
ls

# 7. 进入 os 子目录并运行
cd os
make run
```
