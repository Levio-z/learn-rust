### Tab Autocompletion  Tab 自动补全

Tab 自动补全是一项 shell 功能，可帮助用户通过按 `<TAB>` 键完成命令。

在这个阶段，你将实现两个内置命令的自动补全： `echo` 和 `exit` 。

当用户输入部分命令并按下 `<TAB>` 时，你的 shell 应该：

检查部分文本是否与内置命令（特别是 `echo` 或 `exit` ）的开头匹配。

如果找到匹配项，请补全单词。

在末尾添加一个空格，以便用户可以立即输入参数。

例如，如果用户输入 `ech` 并按下 tab 键，你的 shell 应该将其补全为 `echo` （末尾加一个空格）。
