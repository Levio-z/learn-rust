### 组成：Checksum 计算的数据内容

| 区块         | 说明                                             |
| ---------- | ---------------------------------------------- |
| **伪首部**    | IP 源地址 + 目的地址 + 协议号 + TCP 长度                   |
| **TCP 报头** | 源端口、目的端口、序号、确认号、标志、窗口等字段（注意：**checksum字段设为0**） |
| **TCP 数据** | 即 TCP payload                                  |
