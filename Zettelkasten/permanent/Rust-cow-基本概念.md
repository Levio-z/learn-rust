## 1. 核心观点  

- **`Cow` 是一个智能指针**，用于处理需要延迟克隆（修改）的数据。
	- 延迟克隆：Cow代表了一个“克隆时写入”的模式，也就是说，**只有在数据需要被修改时，它才会进行克隆操作，而在数据没有被修改时，它会保持为借用（只读）**。
- 它支持 **借用数据（`Borrowed`）** 和 **拥有数据（`Owned`）** 两种状态。

## 3. 展开说明  
```
pub enum Cow<B> 
where B: ToOwned + ?Sized { 
Borrowed(&'a B), Owned(B::Owned), 
}
```
- **Borrowed**：表示数据是以借用的方式存在的，数据本身没有被克隆，只有对其的引用。
- **Owned**：表示数据已经被克隆并拥有自己的所有权。
