## 1. 核心观点  

对大多数类型，`clone()` 表示 “**深拷贝**”整个数据结构（Deep Copy ）

## 3. 展开说明  

```rust
let s1 = String::from("hello");
let s2 = s1.clone(); // 深拷贝：复制整段堆内存数据
```

此时：

- `s1` 和 `s2` 各自拥有**独立的数据副本**；
- 复制操作 **会重新分配内存**；
- 拷贝的时间成本与数据大小成正比。

🕓 **深拷贝代价大**，尤其当数据量较大（如大字符串、复杂对象）时。
