### 简化版卡片笔记类型（type）

| 类型（type）       | 说明 / 使用场景                                           |
| -------------- | --------------------------------------------------- |
| 永久笔记           |                                                     |
| **permanent**  | 永久笔记：特别重要的，整合理解后的底层原理知识，形成自己的知识结构而不是不重要的参考资料和规则     |
| **literature** | 文献或资料卡：书籍、文章、论文、视频等来源信息，有成为永久笔记的内容                  |
| **fleeting**   | 临时笔记：灵感、草稿、待整理内容                                    |
| 资料系列           |                                                     |
| **reference**  | 参考资料：资料摘录、文献链接、API 文档、公式等                           |
| 普通笔记           |                                                     |
| **note**       | 普通原子笔记：阅读或实践中提取的原子知识点，输出，提炼组合后的内容，包含大量链接和参考内容，不注重格式 |
| draft          | note的草稿状态                                           |
| toc            | 配合note整理的目录                                         |
| 无标签            | 普通笔记                                                |
