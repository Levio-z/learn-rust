要真正异步化，需要两方面支持：

1. **异步文件**：
    
    - 使用 `tokio::fs::File` 或 `async_std::fs::File`。
        
2. **异步解压**：
    
    - `tokio-util` + `async-compression` 提供异步 Zlib 解压：
        
        `use async_compression::tokio::bufread::ZlibDecoder; use tokio::io::{BufReader, AsyncBufReadExt};  let file = tokio::fs::File::open(path).await?; let decoder = ZlibDecoder::new(BufReader::new(file)); let mut ret = Vec::new(); decoder.read_until(b'\0', &mut ret).await?;`
        
    - 注意这里的 `read_until` 是异步版本，需要 `AsyncBufReadExt`。
