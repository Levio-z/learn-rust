Rust 的目标是既安全又易用
- 为了更好地保障程序的行为，Rust 对代码施加了比 C 和 C++ 更多的限制，而只有靠实践和经验，我们才能逐渐适应这种限制。但就整体而言，这门语言还是灵活且富有表现力的。用 Rust 编写的代码范围之宽、应用领域之广就是证明。
