- **含义：** “超文本”指的是一种非线性的、通过链接（Hyperlinks）互相连接的文本系统。它不仅仅是普通的文字，还包括指向其他文档、图片、视频或任何资源的链接。
	- 非线性：信息以**网状的的方式组织和访问。
	- 链接：超文本使用链接，**组织/引用**其他资源。
    
- **在 HTTP 中的意义：** HTTP 最初被设计出来，就是为了在互联网上高效地传输这些带有链接的文档——主要是 **HTML（HyperText Markup Language，超文本标记语言）** 文件。
    
- **核心功能：** 使得用户能够通过点击链接，在不同的 Web 资源之间跳转和导航，这是 Web 成为一个全球信息网的基础。
