这个命令可以**只暂存文件中的一部分改动**。适用于你在一个文件中修复了多个问题，想要分开提交时使用。
```
git add -p  # 或 --patch
```
你会看到：
```
Stage this hunk [y,n,q,a,d,e,?]?
```

| 输入  | 说明             |
| --- | -------------- |
| `y` | 暂存此块（hunk）     |
| `n` | 跳过此块           |
| `e` | 编辑此块（你可以精细选择行） |
| `q` | 退出交互模式         |
| `?` | 显示帮助           |
