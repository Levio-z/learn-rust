### 2. Windows 系统中的命名管道 (Named Pipes)

Windows 的命名管道语义与 Unix-like 系统的 FIFO 有很大不同。

- **持久性：**
    
    - Windows 命名管道**不是**文件系统中的持久文件。它存在于一个特殊的命名空间 (`\\.\pipe\`) 下。
        
    - Windows 命名管道通常在**最后一个管道句柄被关闭时**（即最后一个连接的服务器或客户端关闭了它的连接）自动销毁。
        
- **内存占用：**
    
    - 命名管道由一个或多个实例组成。
        
    - 当管道实例被创建但没有客户端连接时，它会占用内核资源（用于管理句柄和实例）。
        
    - 一旦最后一个句柄关闭，整个管道就会被销毁并释放其占用的内核内存。
