## 1. 核心观点  
### Ⅰ. 概念层

**业务语义**关注的是：

> “这次修改**意味着什么**，而不仅仅是‘字段被赋了新值’。”

例如：

- 设置默认 Owner ≠ 普通赋值
- 它可能意味着：
    - 系统配置发生变更
    - 新创建的对象将继承新 Owner
    - 需要权限校验 / 审计记录
    - 需要通知其他模块

如果直接写：

```js
defaultOwner = { ... };
```

这条语句**完全丢失了这些语义**。
