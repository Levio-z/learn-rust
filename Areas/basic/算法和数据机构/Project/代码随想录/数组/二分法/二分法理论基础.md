- 前提：**数组为有序数组**，同时题目还强调**数组中无重复元素**

>因为一旦有重复元素，使用二分查找法返回的元素下标可能不是唯一的

- 写法：二分查找是一个循环算法，循环的关键就是每次循环结束都要保持[循环不变量](../../../../Area/inbox/循环不变量.md)为真，每一次对区间的定义都要满足以下循环不变量。
$$\text{循环不变量: 如果目标值 } v \text{ 存在于数组 } A \text{ 中，那么 } v\text{ 一定位于子数组 } A[low \dots high] \text{ 的范围内。}$$
