## 1. 核心观点  
元组解构赋值是指：

```
let (a, b, c) = (1, 2, 3);
```

- 左边是一个 **元组模式**（pattern） `(a, b, c)`
- 右边是一个 **元组值** `(1, 2, 3)`
- Rust 会 **按顺序将右边元素绑定到左边变量**

可以加 `mut` 来声明可变性：

```
let (mut x, mut y) = (10, 20);
x += 1; // 合法
```

## 3. 展开说明  
### .特点

1. **一行初始化多个变量**
    - 避免多行 `let`
2. **可以和模式匹配结合**
    - 更复杂的结构（元组、数组、枚举）也可以解构
3. **类型可以省略或写在右边**
    - 左边模式不能写类型（`let (mut a: i32, mut b: i32)` 会报错）
    - 类型可以写在右边字面量或表达式上：
```
let (mut left, mut right) = (0i32, (n-1) as i32);
```
