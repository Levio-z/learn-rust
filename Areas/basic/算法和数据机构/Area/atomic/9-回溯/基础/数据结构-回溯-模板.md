## 3. 展开说明  
- 回溯函数模板返回值以及参数
    - 函数起名字为backtracking
    - 函数返回值一般为void
    - 伪代码
        ```Plain
        void backtracking(参数)
        ```
- 回溯函数终止条件
    - 树中就可以看出，一般来说搜到叶子节点了，也就找到了满足条件的一条答案，把这个答案存放起来，并结束本层递归。
    - 伪代码
        
        ```Java
        if (终止条件) {
            存放结果;
            return;
        }
        ```
        
- 回溯搜索的遍历过程
    
    - 在上面我们提到了，回溯法一般是在集合中递归搜索，集合的大小构成了树的宽度，递归的深度构成的树的深度。
        ![image.png](https://raw.githubusercontent.com/Levio-z/MyPicture/main/img/20260117211839963.png)

        
            注意图中，我特意举例**集合大小和孩子的数量**是相等的！
        
    - 伪代码
        ```Plain
        for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
            处理节点;
            backtracking(路径，选择列表); // 递归
            回溯，撤销处理结果
        }
        ```
        
- 总体模板
    
    ```Plain
    void backtracking(参数) {
        if (终止条件) {
            存放结果;
            return;
        }
        for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
            处理节点;
            backtracking(路径，选择列表); // 递归
            回溯，撤销处理结果
        }
    }
    ```
