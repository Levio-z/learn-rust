## 3. 展开说明  

1、求索引a-b的区间和使用符号sum(a,b)表示，原始数组使用nums
区间和就可以表示为：
`sum(a,b) = nums[b]+nums[b-1]+..+nums[a]`

2、累加和使用p表示

索引在b的累加和就是从索引0到b所有元素相加，即
`p[b] =nums[b]+nums[b-1]+..+nums[a]+nums[a-1]+..+nums[0]`

3、化简：
`p[b] =nums[b]+nums[b-1]+..+nums[a]`+`+nums[a-1]+..+nums[0]`

使用区间和和累加和替换:
`p[b] = sum(a,b) +p[a-1]`
`sum(a,b) =p[b]-p[a-1]`

假设a = 0；
```
p[a-1]=p[-1]=0
```
  
所以总体可以`p[0]`就是没有元素的情况，`p[1]`就是有一个元素的情况

```
索引b的前缀和，就是p[b+1],因为索引为b，到0的元素有b+1个
```


```
求区间和`sum[a,b]`也就是求，p[b+1]p-[(a-1)+1]
```


4、总结

只需要先求出累加和，后续的区间和运算的时间复杂度都为O(1)
