## 3. 展开说明  

### bitter 的语义层次

1. **味觉层面（literal）**
    
    - 苦味的食物或饮料：  
        _The medicine tastes bitter._  
        _Bitter chocolate is an acquired taste._
        
2. **情感层面（figurative / metaphorical）**
    
    - 表达痛苦或难以接受的经历：  
        _She felt bitter about the betrayal._
        
    - 表示悲伤、憎恨或失望的情绪：  
        _He harbored a bitter resentment._
        
3. **行为或态度（figurative / metaphorical）**
    
    - 尖刻、激烈、强烈：  
        _bitter criticism_ → 尖刻的批评  
        _bitter rivalry_ → 激烈的竞争
### 三、常见搭配

| 搭配                           | 含义        |
| ---------------------------- | --------- |
| bitter taste / bitter flavor | 苦味        |
| bitter truth                 | 残酷的事实     |
| bitter disappointment        | 深深的失望     |
| bitter rival / bitter enemy  | 势不两立的对手   |
| bitter tears / bitter sorrow | 悲痛之泪 / 悲伤 |
