- **6.1 初始配置：** 新设备首次联网如何获取 DNS IP 地址？
    
    - **DHCP 协议：** 通过 **DHCP DISCOVER 广播**，从 DHCP 服务器处动态获取 IP 地址、子网掩码、网关以及 **本地 DNS 服务器的 IP 地址**。
	    - [路由器是怎么获取到DHCP](reference/路由器是怎么获取到DHCP.md)
        
- **6.2 根服务器的 IP 变化：**
    
    - 根服务器地址不绝对固定，**Root Hints 文件需要定期更新**。
        
    - 利用 **Anycast** 技术，单个根服务器 IP 对应全球多个物理服务器实例（提高冗余和响应速度）。
