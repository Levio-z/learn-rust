- **3.1 根 DNS 服务器 (Root Name Servers)：**
    - **数量：** 逻辑上 13 组（A 到 M）。
    - **作用：** 存储所有顶级域服务器的地址，指明下一步去哪里查询。
    - **自举机制：** 所有 DNS 解析器内置**根提示文件 (Root Hints File)**，包含这 13 组服务器的名称和 IP 地址。
- **3.2 顶级域 (TLD) DNS 服务器：**
    - **作用：** 负责管理所有该顶级域下的二级域服务器地址。
- **3.3 权威 DNS 服务器 (Authoritative Name Servers)：**
    - **作用：** 存储特定域名（如 `server.com`）的**最终解析记录**。是域名解析结果的最终出处。
- **3.4 本地 DNS 服务器 / 递归解析器 (Local DNS Resolver)：**
    - **作用：** 接收客户端（浏览器/操作系统）的查询请求，并代替客户端向其他 DNS 服务器发起迭代查询。负责缓存查询结果。
