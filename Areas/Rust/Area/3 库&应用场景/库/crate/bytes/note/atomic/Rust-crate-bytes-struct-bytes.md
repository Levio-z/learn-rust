## 1. 核心观点  
### Ⅰ. 概念层

`Bytes` 是一个**不可变字节视图**，支持：

- O(1) clone
- **零拷贝 slice / split**
- **跨线程安全共享**（`Send + Sync`）
```rust
use bytes::Bytes;

let b1 = Bytes::from_static(b"hello world");
let b2 = b1.clone();
let b3 = b1.slice(0..5);
```
以上操作**不会发生任何内存拷贝**。
