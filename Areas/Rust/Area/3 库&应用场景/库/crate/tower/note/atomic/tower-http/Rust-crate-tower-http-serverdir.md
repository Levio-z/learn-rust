## 1. 核心观点  
### Ⅰ. 概念层

- 功能特性:
    - 可服务指定目录及其子目录下的所有文件
    - 自动根据文件扩展名猜测Content-Type
    - 文件不存在时返回404 Not Found
- 配置选项:
    - append_index_html_if_directory(true)：为目录请求自动附加index.html
    - precompressed_gzip()：支持gzip压缩版本文件
    - precompressed_br()：支持brotli压缩版本文件
    - precompressed_zstd()：支持zstd压缩版本文件
