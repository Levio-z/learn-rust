## 1. 核心观点  
### Ⅰ. 概念层
- 基本介绍
    - 定位: 基于tower框架构建的HTTP中间件集合
    - 功能: 提供HTTP客户端和服务器的各种实用中间件
    - 兼容性: 使用http crate作为HTTP接口，可与任何使用http的库/框架兼容(如hyper)
- 功能
	- Trace: 提供请求和响应的高级日志记录，支持常规HTTP请求和gRPC
    - Compression/Decompression: 响应体的压缩和解压缩功能
    - FollowRedirect: 自动处理重定向响应
    - 其他中间件: 还包括CORS处理、授权验证等实用功能
