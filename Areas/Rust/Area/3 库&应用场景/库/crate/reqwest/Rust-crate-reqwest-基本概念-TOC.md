## 1. 核心观点  
### Ⅰ. 概念层
reqwest crate 提供了一种便捷的、更高级别的 HTTP 服务。
它处理了很多大多数人期望 HTTP 客户端为他们做的事情。
- 异步和[阻塞](https://docs.rs/reqwest/latest/reqwest/blocking/index.html)客户端
- 纯文本、 [JSON](https://docs.rs/reqwest/latest/reqwest/#json) 、 [URL 编码](https://docs.rs/reqwest/latest/reqwest/#forms) 、 [多部分文本](https://docs.rs/reqwest/latest/reqwest/multipart/index.html "mod reqwest::multipart")
- 可自定义[重定向策略](https://docs.rs/reqwest/latest/reqwest/#redirect-policies)
- Uses [TLS](https://docs.rs/reqwest/latest/reqwest/#tls) by default
- HTTP [Proxies](https://docs.rs/reqwest/latest/reqwest/#proxies)  HTTP [代理](https://docs.rs/reqwest/latest/reqwest/#proxies)
- 提供Cookie存储功能
- 支持WASM
