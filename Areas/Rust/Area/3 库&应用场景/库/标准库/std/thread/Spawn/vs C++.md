- **入参**：
    - 类型安全，由模板和 `std::decay_t` 实现。
    - 可按值或按引用传递，但引用需手动管理生命周期。
    - 底层通过 `_Invoker` 对象封装所有参数到堆上，以便操作系统线程调用。
- **返回值**：
    - 不直接返回线程函数结果。
    - 可以通过 `std::promise/std::future` 获取返回值。
    - 异常未捕获则 `std::terminate()`。
- **安全与灵活性**：
    - C++ 提供模板封装，避免裸指针和 `void*`。
    - 类型安全编译期检查，但线程生命周期与引用安全仍由开发者保证。
    - 对比 Rust `thread::spawn`：
        - Rust 编译器强制 `'static` 或 `move` 捕获，保证线程安全。

        - Rust 返回值通过 `JoinHandle<T>` 安全获取。
