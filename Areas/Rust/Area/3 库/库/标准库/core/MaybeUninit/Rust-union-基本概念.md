## 1. 核心观点  

在 Rust 中，`union` 并不是来自某个外部 crate，而是 **Rust 语言的核心语法**，**直接由编译器支持**。你可以把它理解为一种类似 `struct` 的类型，但它**允许多个字段共享同一块内存**。

- **语法定义**：位于 Rust 语言核心（`rustc`）中，不在 `std` 或 `core` 里实现额外逻辑。

## 3. 展开说明  

#### 特点
1. `union` 本身是语言层面的原语，不需要任何 `use` 引入。
2. 所有字段共用同一块内存，访问字段时需小心，通常配合 `unsafe`。
3. `union` 的安全使用场景：
    - FFI（与 C 的数据交互）
    - 内存复用（如 `MaybeUninit`）
    - 构建低层数据结构
