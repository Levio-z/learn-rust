- `PartialEq` trait 
	- 用于支持 **相等性比较 (`==` / `!=`)**。

- `Eq` 是一个标记 trait，继承自 `PartialEq`
	- 它表示 **等式是完全对称和确定的**，没有“不确定相等”情况。

- 枚举、整数为什么可以实现 Eq？
- 集合类型的要求
	- 某些集合类型要求 key 可以严格比较相等：
	    - `HashMap<K, V>`
	    - `HashSet<T>`
	- 这些集合内部依赖“完全对称、确定的相等性”来保证正确性。
	- 如果只实现 PartialEq，但不能实现 Eq，就不能作为这些集合的 key。
