使用File来描述和操作一个文件

### 打开一个文件


#### 文件->字符串
```
pub fn read_to_string<P: AsRef<Path>>(path: P) -> Result<String>
```
>这是使用 [`File::open`](https://rustwiki.org/zh-CN/std/fs/struct.File.html#method.open "associated function std::fs::File::open") 和 [`read_to_string`](https://rustwiki.org/zh-CN/std/io/trait.Read.html#method.read_to_string "method std::io::Read::read_to_string") 且导入次数较少且没有中间变量的便捷函数。

如果 `path` 还不存在，则此函数将返回错误。 根据 [`OpenOptions::open`](https://rustwiki.org/zh-CN/std/fs/struct.OpenOptions.html#method.open "method std::fs::OpenOptions::open")，可能还会返回其他错误。

如果在读取 [`io::ErrorKind::Interrupted`](https://rustwiki.org/zh-CN/std/io/enum.ErrorKind.html#variant.Interrupted "variant std::io::ErrorKind::Interrupted") 以外的其他错误时遇到错误，或者文件的内容不是有效的 UTF-8，它也会返回错误。
