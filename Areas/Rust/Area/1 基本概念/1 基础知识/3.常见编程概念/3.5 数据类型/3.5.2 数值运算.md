Rust 支持您期望的所有数字类型的基本数学运算：加法、减法、乘法、除法和余数。整数除法将截断为零到最接近的整数。
```rust
fn main() {
    // addition
    let sum = 5 + 10;

    // subtraction
    let difference = 95.5 - 4.3;

    // multiplication
    let product = 4 * 30;

    // division
    let quotient = 56.7 / 32.2; // 1.7608695652173911
    let truncated = -5 / 3; // Results in -1

    // remainder
    let remainder = 43 % 5;// 3
    println!("{remainder}");
    let remainder = -43 % 5; // -3
    println!("{remainder}");
}
```
