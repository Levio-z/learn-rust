### 定义

指的是通过使用相同的变量名称并再次声明该变量，从而使得新的声明覆盖或“遮蔽”了先前的变量。
- 涉及内存对象的再分配
- 而 `mut` 声明的变量，可以修改同一个内存地址上的值，并不会发生内存对象的再分配，性能要更好。
### 作用

- 变量遮蔽的用处在于，**如果你在某个作用域内无需再使用之前的变量（在被遮蔽后，无法再访问到之前的同名变量），就可以重复的使用变量名字，而不用绞尽脑汁去想更多的名字。**

### 变量遮蔽的工作原理

- **遮蔽**是通过使用相同的变量名称并用 `let` 关键字重新声明来实现的。
- **在同一作用域内，新的变量声明会“遮蔽”旧的变量**，因此任何使用该名称的地方将使用新的变量。
- **作用域**结束时，遮蔽的变量将不再可用，之前的变量名称将恢复作用。
- 我们再次使用`let`关键字时，我们**实际上是在创建一个新变量，所以我们可以更改值的类型，但重复使用相同的名称。**
```rust
fn main() {
    let x = 5;  // 第一个 x
    println!("The first x is: {}", x);  // 打印 5

    let x = x + 1;  // 第二个 x，遮蔽了第一个 x
    println!("The second x is: {}", x);  // 打印 6

    {
        let x = x * 2;  // 在内部作用域中再次遮蔽 x
        println!("The third x is: {}", x);  // 打印 12
    }

    println!("The second x after inner scope is: {}", x);  // 打印 6
}
```
