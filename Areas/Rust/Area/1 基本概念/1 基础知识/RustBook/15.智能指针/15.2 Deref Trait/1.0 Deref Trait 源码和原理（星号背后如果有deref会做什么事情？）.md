## 1. 核心观点  
```
pub trait Deref {
    type Target: ?Sized;

    fn deref(&self) -> &Self::Target;
}
```

关键点：
- `Target: ?Sized` 允许目标类型是动态大小类型（DST），例如 `str`、trait object。
- 返回值只能是 `&Target`，不能返回 `&mut Target`（可变解引用由 `DerefMut` 负责）。
### 底层原理
**当写 `*x` 时，编译器展开为，Rust 在幕后实际上运行了这段代码：**
```rust
*x.deref()
```
所以如果
```
*String =>*String.deref =>*(&str)=>str
```

- 需要注意的是，**`*` 不会无限递归替换，从 `*y` 到 `*(y.deref())` 只会发生一次**，而不会继续进行替换然后产生形如 `*((y.deref()).deref())` 的怪物。
