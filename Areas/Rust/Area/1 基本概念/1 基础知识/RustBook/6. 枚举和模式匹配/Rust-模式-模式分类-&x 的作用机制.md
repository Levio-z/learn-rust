## 3. 展开说明  
```rust
fn main() {
    let a = 10;
    let b = &a;
    let &x = b; // 模式解构
}
```

编译器行为分两步理解：

1. 检查右值 `b` 的类型是 `&i32`；
    
2. 模式左边 `&x` 表示：
    
    - “匹配一个引用”
        
    - 并绑定其中被引用的值给 `x`
        

此时 `x` 是一个值复制（`Copy` 类型直接拷贝），  
若是非 `Copy` 类型，会发生**所有权移动**或报错。
