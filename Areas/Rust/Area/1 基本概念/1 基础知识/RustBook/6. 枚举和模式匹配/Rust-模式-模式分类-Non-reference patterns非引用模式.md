## 1. 核心观点  

哪些模式 **不算** 非引用模式（即被排除的四类）,除了这些之外都是非引用模式

| 排除的模式类型                    | 示例                              | 解释                                   |
| -------------------------- | ------------------------------- | ------------------------------------ |
| **1. 绑定模式**                | `let ref x = value;`            | 显示使用ref ref mut 或mut绑定               |
| **2. 通配符模式 `_`**           | `match v { _ => ... }`          | `_` 不绑定任何变量，没有绑定行为，不需要自动借用。          |
| **3. 引用类型常量模式**            | `match r { &CONST_REF => ... }` | 这是显式匹配引用常量，类型已经是引用。                  |
| **4. 引用模式 (`&` / `&mut`)** | `match r { &Some(v) => ... }`   | 已经手动写了 `&` 或 `&mut`，编译器不会再自动加 `ref`。 |
