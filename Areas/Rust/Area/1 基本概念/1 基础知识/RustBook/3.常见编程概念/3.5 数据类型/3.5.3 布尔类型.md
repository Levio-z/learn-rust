`bool`

- **`&&` (逻辑与)**
    
    用于检查两个布尔表达式是否都为`true`。如果两个操作数都为`true`，则返回`true`；否则返回`false`。
    
    ```rust
    let a = true;
    let b = false;
    let result = a && b; // result为false
    
    ```
    
- **`||` (逻辑或)**
    
    用于检查两个布尔表达式是否至少有一个为`true`。如果其中一个或两个操作数为`true`，则返回`true`；否则返回`false`。
    
    ```rust
    let a = true;
    let b = false;
    let result = a || b; // result为true
    
    ```
    
- **`!` (逻辑非)**
    
    用于反转布尔值。如果操作数是`true`，则返回`false`；如果操作数是`false`，则返回`true`。
    
    ```rust
    let a = true;
    let result = !a; // result为false
    
    ```
    

与大多数其他编程语言一样，Rust 中的布尔类型有两个可能的值： `true`和`false` 。布尔值的大小为一字节。 Rust 中的布尔类型是使用`bool`指定的。例如：

```rust
fn main() {
    let t = true;

    let f: bool = false; // with explicit type annotation
}
```
