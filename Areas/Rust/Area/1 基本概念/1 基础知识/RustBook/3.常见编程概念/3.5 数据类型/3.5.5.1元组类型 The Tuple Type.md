_元组_是**将具有多种类型的多个值组合**在一起形成一个复合类型的通用方法。元组有固定的长度：一旦声明，它们的大小就不能增长或缩小。元组中每个位置都有一个类型。对于从函数返回的多个值特别有用。

- 声明
    
    - `let tup: (i32, f64, u8) = (500, 6.4, 1);`
- 访问元素，索引从0开始
    
    - 句点 ( `.` ) 后跟要访问的值的索引来直接访问元组元素。
    - 修改元组，声明为mut即可，然后通过句点来修改
- 模式匹配：解构
    
    - **let (x, y, z) = tup;**
- 示例如下
    
    - 解构
        
        ```rust
        fn main() {
            let tup: (i32, f64, u8) = (500, 6.4, 1);
            
            let (x, y, z) = tup;
        
            println!("The value of y is: {y}");
        }
        ```
        
        该程序首先创建一个元组并将其绑定到变量`tup` 。然后，它使用带有`let`模式来获取`tup`并将其转换为三个单独的变量`x` 、 `y`和`z` 。这称为_解构_，因为它破坏了 将单个元组分成三个部分。
        
    - 句点
        
        ```rust
        fn main() {
            let x: (i32, f64, u8) = (500, 6.4, 1);
        
            let five_hundred = x.0;
        
            let six_point_four = x.1;
        
            let one = x.2;
        }
        ```
        
        没有任何值的元组有一个特殊的名称， _unit_ 。个值和它对应的类型都写成`()` ，表示一个空值或者一个空返回类型。如果表达式不返回任何其他值，则它们隐式返回单位值。
        
- 修改元组
    
    ```rust
    fn main() {
        let mut x: (i32, i32) = (1, 2);
        x.0 = 0;
        x.1 += 5;
    }
    ```
