## 1. 核心观点  
**泛型参数 trait bound**
    
- 函数签名：

	```rust
	fn write<C: AsRef<[u8]>>(contents: C) {}
	```
**泛型参数 trait bound 是类型约束**
    
- Rust 编译器要求 **明确实现** trait
- 如果泛型参数没有直接实现 trait，Rust **不会自动尝试 deref 内部类型**
