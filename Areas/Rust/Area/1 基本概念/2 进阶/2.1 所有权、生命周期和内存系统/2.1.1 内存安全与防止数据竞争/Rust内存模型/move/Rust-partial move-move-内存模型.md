### 初始设置：字段 move 的特殊性（partial move）

`let y = n.next;`

MIR 里是：
```
move n.next -> y 
drop_flag(n.next) = false
```

此时：
```
n.val  : initialized
n.next : UNINIT
```

所以：

- `n` 整体不可用
- 但 `n.val` 仍可单独访问
