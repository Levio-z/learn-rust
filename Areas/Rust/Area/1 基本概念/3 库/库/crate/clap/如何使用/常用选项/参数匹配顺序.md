| 参数类型                                        | 优先级   | 说明                       |
| ------------------------------------------- | ----- | ------------------------ |
| 带前缀（`--` / `-`）的命名参数                        | ★★★★★ | 在命令行任意位置均会被优先识别          |
| 子命令（`#[command(subcommand)]`）               | ★★★★☆ | 一旦匹配到已注册子命令名，就立即停止位置参数匹配 |
| 位置参数（`positional arguments`）                | ★★☆☆☆ | 按顺序消费输入                  |
| 剩余参数（`#[arg(last = true)]` 或 `Vec<String>`） | ★☆☆☆☆ | 最后才接收剩余参数                |
